## PC SPC

1. SPC 查询条件，只需要时间范围（默认查询一周的），物料号、检测项、批号范围（开始-结束）（只需要 QC 首检、巡检模板的检查项进行SPC的处理计算（qc 首检、巡检，模板中的检查项是统一的））√
2. SPC 的数据源只需要 QC检测定量部分检查数据  √
3. 按照批次进行分组（一般两个，封顶5个，（注意：每组数量不固定））为一组进行数据处理 √
4. 制成能力，只有满处25组数据以上才可以计算CPK、PPK √





## 质量大屏

1. 检测模板需要加一个类型特性（SPC 特性尺寸，用来大屏轮播显示检查项用）

2. 多个检查尺寸的轮播时间 1-2 分钟，需要多个物料、以及对应的检查尺寸进行轮播，一次轮播一个检查尺寸的 XBar、Rbar Chart 图
3. 需要进行轮播的检查项是，是现场设备正在生产的物料（正在生产中设备对应的物料对应的检测尺寸（检测项）） √
4. 需要加一个领导模式（应对检查，领导来检查时,不合格、不良、不通过的等都不显示出来） √



## PAD 三坐标

1.模板他们的意思可以由我们来定（按照功能来确定膜拜你）

2.物料多，没有物料三坐标导出的检测项不固定（数量不统一）（并且可以检查到的检查项不能全部填充检查模板内容）

3.目的是需要减少录入时间；

4.需要回去讨论可行性，以及方案在说
